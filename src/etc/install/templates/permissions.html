Furnace Permissions

Furnace needs to be able to write to certain files and directories inside your installation. If any of the indicators below are not green, please take the following steps to ensure the files/directories are writeable:

Hint:

if your web server is running as user 'www-data', and the directory exists at 'path', then from the command line, enter the following two commands to make the directory writeable:
chgrp -R www-data 'path'  and then
chmod -R g+w 'path'
