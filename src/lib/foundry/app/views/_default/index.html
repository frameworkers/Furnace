Welcome to Furnace!

We hope you will find that Furnace helps you build out your application quickly and easily. Furnace is still developing, and we would greatly appreciate your feedback. If you find something confusing, poorly documented, or just plain broken, please let us know here so that we can continue to improve this platform.

Project Details:

The installed root directory for your project appears to be:
[_furnace.rootdir]
Application settings are located in: [rootdir]/app/config/app.yml
Controllers are located in: [rootdir]/app/controllers/
Views are located in: [rootdir]/app/views/

Getting Started:

As with any framework, Furnace has a few basic principles which, once understood, will allow you to take full advantage of its power. The latest documentation is always available online along with FAQs, tutorials, and other helpful material.
