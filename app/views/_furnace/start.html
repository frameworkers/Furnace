Welcome to Furnace!

The project root directory is: [_furnace.rootdir]
Application settings are located in: [rootdir]/app/config/app.yml
Controllers are located in: [rootdir]/app/controllers/
Views are located in: [rootdir]/app/views/
To make changes to this page, edit: [rootdir]/app/views/_default/index.html
To manage your application's model & data, first set a root password in [tp.rootdir]/app/config/app.yml and then visit Fuel. Also, take a moment to learn more about the Tadpole PHP template engine built into Furnace.
