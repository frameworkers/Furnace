Furnace is a rapid application development framework for PHP 5.3+ web applications. It provides a fast, flexible framework for developing applications without restriction.

Furnace is open source software, publicly released under the M.I.T. license. Development on Furnace is collaborative, and we warmly welcome contributions of any kind. Please look at the community guidelines and our how-to for developers interested in moving Furnace forwards.
